## 📁 Organização do Repositório

O projeto está dividido para facilitar a manutenção e o entendimento da estrutura:

* **DDL:** Definição da estrutura do banco (Tabelas, Chaves Primárias e Estrangeiras).
* **DML:** Scripts de população inicial para testes.
* **DQL:** Consultas complexas utilizando diversos tipos de `JOINs` para extração de relatórios.

## 🚀 Como Executar

1. Execute o script em `scripts/01_ddl_schema.sql` para criar o banco de dados.
2. Popule as tabelas com `scripts/02_dml_inserts.sql`.
3. Utilize os comandos em `scripts/03_dql_queries.sql` para visualizar os resultados.
